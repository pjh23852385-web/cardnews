# stripe — 프리미엄 비즈니스 톤

> 어울리는 오디언스: 경영진, 임원, C-Level, 투자자, B2B 고객
> 어울리는 주제: 금융, 전략, KPI 리포트, 파트너십, 재무 분석

## 영감 출처
- **레포**: [github.com/VoltAgent/awesome-design-md](https://github.com/VoltAgent/awesome-design-md) → `stripe/DESIGN.md`
- **원본 브랜드**: Stripe (signature purple gradients with weight-300 elegance)

## 핵심 색감
- **배경**: `#ffffff` 또는 `#f6f9fc` (very light blue-gray)
- **주 텍스트**: `#0a2540` (deep navy, 텍스트 중심)
- **보조 텍스트**: `#425466` (slate gray)
- **액센트**: `#635bff` (Stripe purple / indigo)
- **그라디언트**: `linear-gradient(135deg, #635bff 0%, #00d4ff 100%)` (퍼플 → 시안)
- **구분선**: `#e3e8ee`

## 폰트
- **헤드라인**: `Pretendard Variable` weight 300 (thin + large, 프리미엄 느낌)
- **본문**: `Pretendard Variable` weight 400
- **숫자**: `Pretendard Variable` weight 500, tabular-nums

## 인터랙션 느낌
- 부드러운 슬라이드 (ease-out)
- 퍼플 그라디언트가 움직이는 히어로 배경
- 카드 hover는 살짝 올라가는 느낌 (translateY -2px + shadow)
- 여백 넉넉히

## 무드 키워드
premium / trustworthy / financial / elegant / corporate-but-modern

## 이 스타일에 어울리는 이미지
- 퍼플 그라디언트 추상 배경
- 금융/데이터 시각화 (미니멀 그래프)
- 기하학적 일러스트 (도형 조합)
- 블루/퍼플 톤의 아이소메트릭

## 이 스타일에 어울리지 않는 것
- 검정 배경 + 네온
- 레트로 매거진 느낌
- 거친 텍스처
- 경고 색 (빨강/노랑)
