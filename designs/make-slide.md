# make-slide — 프레젠테이션 멀티테마 갤러리

> 어울리는 오디언스: B2B, 스타트업 피치, 기술 컨퍼런스, 교육, 경영진 보고
> 어울리는 주제: 프레젠테이션형 콘텐츠, 슬라이드 덱, 데이터 분석, 비즈니스 인사이트

## 영감 출처
- **URL**: [make-slide.vercel.app](https://make-slide.vercel.app)
- **특징**: 단일 스타일이 아닌 **여러 테마 + 레이아웃 조합** 갤러리. 프레젠테이션(슬라이드 덱) 전용 레퍼런스

## 핵심 색감 (테마별 다름 — 대표 조합)
- **다크 모드 베이스**: 검정 배경 + 흰색 텍스트
- **라이트 모드 베이스**: 흰색 배경 + 검정 텍스트
- **테마 예시**:
  - `Neon Terminal`: 다크 + 네온 그린/시안 (voltagent 유사)
  - `Corporate`: 네이비 + 골드
  - `Gradient Pop`: 퍼플 → 오렌지 그라디언트
  - `Editorial`: 매거진 톤 (wired 유사)

## 폰트
- **헤드라인**: 굵은 산세리프 (대비 강하게)
- **본문**: 클린·가독성 우선
- 특정 폰트 지정 없음 → 기본 Pretendard Variable 쓰면 됨

## 레이아웃 (4종 — 상황별 선택)
| 레이아웃 | 언제 |
|---|---|
| **Centered** (중앙 정렬) | 표지, 임팩트 슬라이드 |
| **Wide** (전폭) | 히어로 이미지, 챕터 구분 |
| **Split** (2컬럼) | 데이터+설명, 비교 슬라이드 |
| **Editorial** (매거진형) | 긴 텍스트, 심층 설명 |

## 인터랙션 느낌
- 슬라이드 넘김 중심 (카드 덱 느낌)
- 과한 애니메이션 X, 깔끔하게
- 다크/라이트 토글 지원하면 좋음

## 무드 키워드
professional / minimal / versatile / presentation-first / B2B-friendly

## 이 스타일에 어울리는 이미지
- 데이터 시각화 (차트, 그래프)
- 아이소메트릭 일러스트
- 추상 기하학 패턴
- 로고/브랜드 자산

## 이 스타일에 어울리지 않는 것
- 캐주얼한 이모지 남발
- 손그림 일러스트
- 과한 그라디언트 (테마 선택에 따라 예외)

## 활용 팁
이 레퍼런스는 **"레이아웃 4종 중 어느 걸 쓸지"** 고를 때 특히 유용.
아트디렉터가 3옵션 제안할 때 이 갤러리에서 한두 개 조합 꺼내 쓸 수 있음.
예: "make-slide의 Editorial + Corporate 테마 × Linear 인터랙션"
