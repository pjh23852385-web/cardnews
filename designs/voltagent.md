# voltagent — AI/Tech 톤

> 어울리는 오디언스: 개발자, 기술 관리자, AI/Tech 관심 경영진
> 어울리는 주제: AI, LLM, 에이전트, 자동화, 개발 툴, 인프라

## 영감 출처
- **레포**: [github.com/VoltAgent/awesome-design-md](https://github.com/VoltAgent/awesome-design-md) → `voltagent/DESIGN.md`
- **원본 브랜드**: VoltAgent (void-black canvas with emerald accent, terminal-native)

## 핵심 색감
- **배경**: `#000000` (void black) 또는 `#0a0a0a` (very dark)
- **주 텍스트**: `#ffffff` (순백)
- **보조 텍스트**: `#a0a0a0` (회색)
- **액센트**: `#00ff7f` (emerald green / spring green) — 글로우 효과에 사용
- **보조 액센트**: `#00d66a` (약간 더 진한 에메랄드)
- **구분선/박스**: `rgba(0, 255, 127, 0.15)` (에메랄드 + 투명)

## 폰트
- **헤드라인**: `JetBrains Mono` weight 700 (터미널 미학, 강한 인상)
- **본문**: `Pretendard Variable` weight 400-600 (한글 가독성)
- **숫자/코드**: `JetBrains Mono` weight 500 (고정폭)

## 인터랙션 느낌
- 터미널처럼 커서 깜빡임 효과
- 에메랄드 글로우 (box-shadow) 버튼/카드 hover
- 슬라이드 전환은 fade (cube 같은 화려한 효과 X)

## 무드 키워드
technical / futuristic / precise / minimalist / hacker-aesthetic

## 이 스타일에 어울리는 이미지
- 추상적 네트워크 시각화
- 회로 / 노드 / 그래프 일러스트
- 검정 배경 + 에메랄드 글로우
- 텍스트 없는 미니멀 배경 일러스트

## 이 스타일에 어울리지 않는 것
- 따뜻한 색 (오렌지, 파스텔)
- 서브틀하지 않은 그라디언트
- 인물 사진
- 화려한 애니메이션
