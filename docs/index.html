# 직무 맞춤 설계 # 환경 개선 # 조직 공감

한화생명금융서비스·한화손해보험은 네일관리사 직무를 도입해 청각장애 직원이 안정적으로 근무할 수 있는 환경을 조성했다. 한화손해보험은 장애인 디자이너가 재택근무로 마케팅 콘텐츠를 제작하도록 지원해 경력 단절을 최소화했고, 한화투자증권은 어학강사 보조 직무로 임직원 글로벌 역량 강화까지 연결했다. '원팀 플러스 데이' 문화 체험 프로그램과 근무 경험 공유 행사를 통해 조직 공감도 함께 끌어올리고 있다.

포용 고용은 CSR 보고서용이 아니다. 직무 설계→환경 개선→조직 공감 3단계를 순서대로 밟아야 실질적 성과와 조직 몰입이 동시에 올라간다. HR전략실과 현업 부서가 공동으로 직무 발굴 TF를 먼저 구성하라
